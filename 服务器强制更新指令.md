# 服务器强制更新指令

## 强制重置并更新（会覆盖服务器上的本地修改）

```bash
cd /www/wwwroot/uniapp && git fetch origin && git reset --hard origin/main && cd backend && pm2 restart uniapp-backend
```

## 分步执行

### 1. 进入项目目录
```bash
cd /www/wwwroot/uniapp
```

### 2. 获取最新代码
```bash
git fetch origin
```

### 3. 强制重置到最新版本（会丢弃服务器上的本地修改）
```bash
git reset --hard origin/main
```

### 4. 重启后端服务
```bash
cd backend && pm2 restart uniapp-backend
```

## 一键执行命令
```bash
cd /www/wwwroot/uniapp && git fetch origin && git reset --hard origin/main && cd backend && pm2 restart uniapp-backend
```
