# 服务器更新指令

## 1. 进入项目目录
```bash
cd /www/wwwroot/uniapp
```

## 2. 拉取最新代码
```bash
git pull origin main
```

## 3. 重启后端服务
```bash
cd backend
pm2 restart uniapp-backend
```

## 或者使用以下一键命令
```bash
cd /www/wwwroot/uniapp && git pull origin main && cd backend && pm2 restart uniapp-backend
```

## 如果拉取失败，强制重置
```bash
cd /www/wwwroot/uniapp
git fetch origin
git reset --hard origin/main
cd backend && pm2 restart uniapp-backend
```
